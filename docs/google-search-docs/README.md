## Quick Reference

### Key Tools

- **[Search Console](https://search.google.com/search-console)** - Monitor your site's performance
- **[Rich Results Test](https://search.google.com/test/rich-results)** - Test structured data
- **[PageSpeed Insights](https://pagespeed.web.dev)** - Analyze page speed
- **[Mobile-Friendly Test](https://search.google.com/test/mobile-friendly)** - Check mobile compatibility

### Core Principles

1. **Create helpful, reliable, people-first content**
2. **Ensure technical requirements are met** (crawlable, indexable)
3. **Avoid spam policies violations**
4. **Optimize page experience** (speed, mobile-friendly, secure)
